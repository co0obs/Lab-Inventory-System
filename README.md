# Lab-Inventory-System
## **Overview**
The Lab Inventory System is a digital platform designed to help laboratories efficiently manage equipment, supplies, and consumables. The system ensures proper tracking of usage, prevents loss or misplacement of items, and provides visibility into stock levels. It also supports role-based access control, ensuring that sensitive or high-value equipment is monitored and managed appropriately.

Features and Functionality
1. Core Inventory System  
   a. Add edit and remove items with a search and filter function  
   b. Track quantities with a categorization system
2. User Tracking  
   a. Check-in/Check-out system for utilized equipment with a potential reservation system  
   b. Track usage history and alerts if items have low storage counts
3. Access Control  
   a. Access differs based on position  
   b. Edit Logs to track who borrowed what and stuff
4. QOL  
   a. Basic GUI  
   b. Easy to use system
